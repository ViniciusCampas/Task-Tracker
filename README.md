ENG - Task Tracker CLI

Task Tracker CLI is a simple task management application developed in Python. The goal of this project is to allow users to manage their tasks 
directly from the terminal, making it possible to add, update, delete, list, and change the status of tasks.

Task information is stored persistently in a JSON file called Task.json, ensuring that data is not lost when the program is closed.

The project was built using Python 3 and only standard libraries such as datetime, json, and sys. This makes the application easy to run and suitable 
for learning purposes.

The application can be used in two different ways: by passing arguments through the command line or by using an interactive mode. When using command-
line arguments, the user can create tasks, update task details, delete tasks, change their status, and list tasks filtered by status.

Each task has a unique identifier (ID), a description, a status (todo, in-progress, or done), as well as creation and last update timestamps.

If the program is executed without any arguments, an interactive menu is displayed in the terminal. In this mode, users can simply choose an option 
by entering the corresponding number, making the application easier to use for beginners.

This project was created with a focus on learning and practicing basic Python concepts such as file handling, data structures, input validation, and 
command-line application development, making it suitable for beginner, junior, or internship-level developers.

PT-BR - Task Tracker CLI

O Task Tracker CLI é uma aplicação simples de gerenciamento de tarefas desenvolvida em Python. O objetivo do projeto é permitir que o usuário 
gerencie suas tarefas diretamente pelo terminal, podendo adicionar, atualizar, excluir, listar e alterar o status das tarefas.

As informações das tarefas são salvas de forma persistente em um arquivo JSON chamado Task.json, garantindo que os dados não sejam perdidos ao 
encerrar o programa.

O projeto foi desenvolvido utilizando Python 3 e apenas bibliotecas padrão da linguagem, como datetime, json e sys. Isso torna a aplicação fácil de 
executar e ideal para fins de aprendizado.

A aplicação pode ser utilizada de duas formas: por meio de argumentos passados diretamente pela linha de comando ou através de um modo interativo. No 
modo de linha de comando, o usuário pode executar comandos para adicionar tarefas, atualizar informações, remover tarefas, alterar o status e listar 
tarefas filtrando pelo status.

Cada tarefa possui um identificador único (ID), uma descrição, um status (todo, in-progress ou done), além da data de criação e da última atualização.

Caso o programa seja executado sem argumentos, um menu interativo é exibido no terminal, permitindo que o usuário escolha as ações digitando apenas o 
número correspondente. Esse modo é ideal para quem está começando a utilizar a aplicação.

Este projeto foi criado com foco em aprendizado e prática de conceitos básicos de Python, como manipulação de arquivos, estruturas de dados, validação de entradas e criação de aplicações de linha de comando, sendo adequado para nível iniciante, júnior ou estágio.
